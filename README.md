# qwerty
Описание веб‑сайта интернет‑магазина электроники
Интернет‑магазин электроники — это полнофункциональный онлайн‑ресурс для продажи электронных устройств и аксессуаров. Сайт разработан с применением современных технологий, обеспечивающих высокую производительность, масштабируемость и удобство поддержки.

Основные функции сайта
Каталог товаров — структурированный список электроники с фильтрацией и сортировкой.

Страница товара — детальное описание каждого устройства: характеристики, фото, цена, наличие.

Корзина покупок — возможность добавлять, удалять и изменять количество товаров.
Адаптивный дизайн — корректное отображение на ПК, планшетах и смартфонах.

Технологический стек
Flask — веб‑фреймворк на Python для реализации серверной логики: обработка запросов, работа с БД, генерация страниц.

Jinja2 — шаблонизатор для динамического формирования HTML‑страниц на основе данных от Flask.

HTML/CSS — разметка и визуальное оформление интерфейса.

JavaScript — интерактивность на клиенте: обновление корзины без перезагрузки страницы, валидация форм, динамическая подгрузка контента.

Docker — контейнеризация приложения для единообразной работы в любой среде (разработка, тестирование, продакшн).

Реализация ключевых компонентов
1. Каталог товаров

Сервер (Flask) получает данные о товарах из БД.

Jinja2 подставляет данные в шаблон HTML, формируя страницу каталога.

JavaScript добавляет интерактивные фильтры и сортировку «на лету».

2. Страница товара

Flask передаёт детальные данные о выбранном товаре.

Jinja2 отображает карточку товара: название, описание, характеристики, галерею фото, цену.

JavaScript реализует:
моментальное добавление товара в корзину
3. Корзина покупок

Клиентская часть (JavaScript):

добавляет/удаляет товары в локальное хранилище (localStorage) или сессионное хранилище (sessionStorage);

обновляет количество и пересчитывает общую сумму;

отправляет AJAX‑запросы на сервер при изменении корзины.

Серверная часть (Flask):

обрабатывает запросы от JS;

синхронизирует корзину с учётной записью пользователя (если авторизован);

сохраняет данные в БД;

возвращает актуальные данные для обновления интерфейса.
